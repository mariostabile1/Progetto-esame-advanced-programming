# Installazione

La web app è già buildata, in `AI-Web-Face-Detector/dist` dove sonp presenti già tutti gli asset e i file per avviarla e basta,
ma se si volesse comunque re-buildare basta andare su AI-Web-Face-Detector/ e scrivere `npm run build`

# Utilizzo

La web app funziona offline e per avviarla andare su `AI-Web-Face-Detector/dist` e avviare un server http con `python3 -m http.server 8000`
(Altre modalità di avvio sono descritte in `Abstract-generati-da-IA/walkthrough.md` direttamente dall'IA)
